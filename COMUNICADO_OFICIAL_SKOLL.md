# Presentación Oficial: Reloj Mundial PatagonIA

Estimados miembros de la comunidad **Skoll**,

Es un placer presentarles **Reloj Mundial PatagonIA**, una herramienta de escritorio diseñada para optimizar la gestión del tiempo y mejorar la productividad en entornos de trabajo globales. Esta aplicación surge de la necesidad de contar con una interfaz clara, estética y funcional para coordinar equipos en distintas zonas horarias.

## Acceso y Distribución

Para facilitar el acceso a todos los integrantes de la comunidad, hemos dispuesto dos modalidades de obtención del software:

### 1. Usuario Final (Ejecutable)
Si desea comenzar a utilizar la aplicación de forma inmediata, puede descargar el instalador **`Reloj de Escritorio Setup 1.0.0.exe`** adjunto en este directorio.
*   **Instrucciones**: Ejecute el instalador en su sistema Windows. Al tratarse de un software independiente, es posible que el sistema solicite confirmación adicional ("Ejecutar de todas formas").

### 2. Desarrollo y Código Fuente (Repositorio GitHub)
Para aquellos interesados en la arquitectura del software o que deseen contribuir al crecimiento del proyecto, el código fuente completo se encuentra disponible bajo licencia de código abierto en:
*   **Repositorio**: [https://github.com/jsarome/reloj-de-escritorio-Patagon-IA]

## Funcionalidades Destacadas
*   **Precisión Global**: Gestión dinámica de zonas horarias con soporte para horario de verano automático.
*   **Foco y Productividad**: Temporizador integrado basado en metodología de bloques de tiempo.
*   **Conversión Horaria**: Herramienta de cálculo para planificación de reuniones internacionales.
*   **Interfaz Glassmorphism**: Diseño minimalista con transparencias que se integra al ecosistema de Windows.
